# Service API Autotests (Postman)

## Запуск тестов

1. Импортируйте коллекцию в Postman
2. Создайте окружение из environment.json
3. Укажите ваш baseUrl в переменной окружения
4. Запустите коллекцию: Run collection

## Тестируемые эндпоинты
- POST /api/service
- GET /api/service/:id  
- PUT /api/service/:id
- DELETE /api/service/:id
